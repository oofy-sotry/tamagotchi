# 다마고치 데스크톱 펫 - 설계 문서

## 개요
- **형태**: Electron 기반 데스크톱 펫 (시메지 스타일)
- **표시 방식**: 투명 프레임리스 창 → 캐릭터만 화면 위에 떠있음
- **항상 최상위**: 다른 창 위에 표시

---

## 기술 스택
- **Electron** (투명 창, 트레이 아이콘)
- **HTML/CSS** (캐릭터 렌더링, 애니메이션)
- **JavaScript** (게임 로직, 상태 관리)

---

## 핵심 기능

### 1. 상태 게이지 (시간 경과에 따라 자동 감소)
- 배고픔
- 행복도
- 체력/건강
- 청결도

### 2. 행동 (우클릭 메뉴로 실행)
- 밥 주기 → 배고픔 회복
- 놀아주기 → 행복도 상승
- 재우기 → 체력 회복
- 씻기기 → 청결도 회복
- 약 주기 → 아플 때 치료
- 쓰다듬기 → 친밀도 상승

### 3. 성장 시스템
- **알** → **아기** → **어린이** → **성체** 단계별 진화
- 돌봄 방식에 따라 다른 모습으로 진화
- 각 단계별 다른 외형/애니메이션

### 4. 감정 표현
- 기쁨, 슬픔, 배고픔, 졸림, 아픔 등 표정 변화
- 상태에 따른 애니메이션 (뛰기, 울기, 자기 등)

### 5. 이동
- 화면 위를 자유롭게 돌아다님
- 드래그로 위치 이동 가능
- 화면 가장자리에서 반응 (벽에 부딪히기 등)

### 6. 똥 시스템
- 주기적으로 똥을 쌈
- 클릭으로 치울 수 있음

### 7. 알림
- 배고프거나 아프면 알려줌

### 8. 저장/불러오기
- 앱 종료 후 재시작해도 상태 유지

### 9. 나이 시스템
- 며칠째 키우고 있는지 표시

### 10. 사망/부활
- 방치하면 사망
- 새로 시작 가능

## 프로젝트 구조 (예정)
```
Tamagotchi/
├── package.json          ← (완료)
├── main.js               ← Electron 메인 프로세스
├── preload.js            ← IPC 통신 브릿지
├── src/
│   ├── index.html        ← 캐릭터 렌더링
│   ├── styles.css        ← 스타일/애니메이션
│   ├── renderer.js       ← UI 상호작용, 메뉴, 드래그
│   └── game.js           ← 게임 로직, 상태, 성장, 저장
├── assets/               ← 캐릭터 이미지/스프라이트 (예정)
```
